连接网络

一、无线网络连接

  1. 搜索可用网络
    在设备的WiFi设置中，搜索SSID为 STNstudio-5G 或 STNstudio-2.4G 的无线网络
  2. 选择最佳频段
    - 5GHz频段（STNstudio-5G）：速度更快，干扰更少，适合新设备
    - 2.4GHz频段（STNstudio-2.4G）：覆盖更广，穿透力强，适合老设备
  3. 输入密码
    密码：STN2025@Studio
    注意：密码区分大小写，建议复制粘贴
  4. 等待获取IP
    连接成功后，设备会自动获取IP地址（DHCP分配）
    如果30秒内未获取到IP，请尝试忘记网络后重新连接

二、有线网络连接

  1. 准备网线
    使用超五类（Cat5e）或以上规格的网线
  2. 连接设备
    将网线一端插入宿舍墙上的网络接口，另一端插入电脑网口
  3. 配置网络
    确保网络设置为自动获取IP地址（DHCP）和DNS服务器
    # Windows配置命令 netsh interface ip set address "以太网" dhcp netsh interface ip set dns "以太网" dhcp # macOS/Linux配置 sudo dhclient en0 # en0为网卡名称

认证登录

一、Web认证（推荐）

  1. 打开浏览器
    连接网络后，打开任意浏览器（Chrome、Edge、Safari等）
  2. 访问任意网站
    在地址栏输入任意网址（如 https://www.baidu.com），会自动跳转到认证页面
  3. 如果未跳转
    手动访问认证地址：http://portal.stnstudio.com
  4. 输入账号密码
    - 账号：您的学号
    - 密码：初始密码为身份证后6位
    首次登录后请立即修改密码！
    建议使用大小写字母+数字+符号组合

二、802.1X认证（高级）

802.1X认证支持无感知登录，一次配置终身使用，推荐高级用户使用

Windows配置步骤：

  1. 打开"设置" → "网络和Internet" → "WLAN"
  2. 点击"管理已知网络" → 选择"STNstudio-5G" → "属性"
  3. 启用"使用IEEE 802.1X认证"，配置如下：
    身份验证方法：PEAP 第二阶段认证：MSCHAPv2 服务器证书：自动 身份（用户名）：您的学号 密码：您的密码

macOS配置步骤：

  1. 在"系统偏好设置" → "网络" → "WiFi"中点击"高级"
  2. 选择"STNstudio-5G"，在"802.1X"选项卡中配置
  3. 输入用户名（学号）和密码，点击"应用"

iOS/Android配置：

连接STNstudio-5G后，在弹出框中输入学号和密码，选择"PEAP"认证方式

常见问题

Q1: 无法连接WiFi，提示"密码错误"

解决方法：
  1. 确认SSID是否为STNstudio-5G或STNstudio-2.4G
  2. 检查密码大小写，建议复制：STN2025@Studio
  3. 尝试"忘记网络"后重新连接
  4. 如仍无效，请联系技术支持重置密码

Q2: 连接成功但无法上网

排查步骤：
  1. 确认是否已完成Web认证或802.1X认证
  2. 检查IP地址是否获取成功（应获取到10.x.x.x网段IP）
  3. 尝试访问http://portal.stnstudio.com强制跳转
  4. 检查DNS设置，建议设置为自动获取或114.114.114.114

Q3: 网速慢、延迟高

优化建议：
  1. 优先连接5GHz频段（STNstudio-5G）
  2. 避开高峰期（19:00-23:00）下载大文件
  3. 检查是否有后台应用占用带宽（如系统更新、云同步）
  4. 尝试重启无线网卡或重新连接
  5. 如持续异常，提交网络诊断报告

Q4: 频繁断线重连

可能原因及解决：
  • 信号强度弱：靠近AP或调整设备位置
  • IP地址冲突：释放并重新获取IP（ipconfig /release && ipconfig /renew）
  • 驱动程序问题：更新无线网卡驱动到最新版本
  • 设备兼容性问题：联系技术支持提供MAC地址进行排查

Q5: 如何修改密码

  1. 访问用户自助服务系统：http://selfservice.stnstudio.com
  2. 使用当前账号密码登录
  3. 在"安全设置" → "修改密码"中设置新密码
  4. 密码要求：至少8位，包含大小写字母、数字和特殊字符

网络优化建议

设备端优化

  • 定期清理系统垃圾，关闭不必要的后台程序
  • 更新网卡驱动到最新版本
  • 使用5GHz频段，减少2.4GHz干扰
  • 设置静态DNS：114.114.114.114 / 223.5.5.5

游戏玩家专用优化

低延迟模式已自动启用
系统已识别主流游戏流量并自动优化，无需手动配置。如需自定义端口，请联系技术支持。

P2P下载优化

限速建议
为保障整体网络质量，P2P下载建议限速为总带宽的70%。

网络安全建议

账号安全

  • 首次登录后立即修改初始密码
  • 不要将账号密码分享给他人
  • 定期更换密码（建议每3个月）
  • 发现异常登录立即修改密码并联系技术支持

设备安全

防病毒建议
安装正版杀毒软件，定期全盘扫描。避免访问可疑网站，不随意下载未知文件。

防ARP攻击

系统已启用ARP防护，如遇到ARP攻击提示，请：

  1. 立即断开网络连接
  2. 使用命令 arp -d * 清除ARP缓存
  3. 重新连接网络并提交攻击报告
